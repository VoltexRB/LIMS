llm_interaction_manager.api.lims_interface.add_persistent_data(conversation, messages)

Adds new persistent data to the database. Data has to contain messages and conversations that align with the internal datastructure. Conversations must contain a conversation_id, messages must each contain a message_id

Parameters:

  • conversation (dict) – Conversation Object with conversation_id that should be inserted

  • messages (list[dict]) – List of messages with message_ids that should be inserted and linked to the conversation
